## anywhere

anywhere是一个静态服务器的神器，使用它我们可以很简单快速的运行本地的dist文件

**使用步骤：**

1.  安装node.js
2.  任意文件夹下全局安装anywhere ：npm install anywhere -g
3.  在工作目录下面输入命令： anywhere
4.  即可完美运行界面，然后再点击index.html



技术栈：

1.  数据迁徙图
2.  异步获取数据
3.  json数据服务端数据封装
4.  json数据客户端解析
5.  echarts动态绑定与行为交互
6.  echarts api使用技巧
7.  图标高级渲染技巧
